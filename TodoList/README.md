# 📝 To-Do List Application

## 📌 Project Description
This is a simple command-line To-Do List application developed using Python. It helps users manage their daily tasks by allowing them to add, view, update, and delete tasks.

## ✨ Features
- Add a new task
- View all tasks
- Update an existing task
- Delete a task
- Exit the application

## 🛠️ Technologies Used
- Python 3.13
- Visual Studio Code

## ▶️ How to Run
1. Open the project folder in VS Code.
2. Open the terminal.
3. Navigate to the project folder:
   ```
   cd TodoList
   ```
4. Run the program:
   ```
   py todo.py
   ```

## 📋 Menu
```
===== TO-DO LIST =====
1. Add Task
2. View Tasks
3. Update Task
4. Delete Task
5. Exit
```
