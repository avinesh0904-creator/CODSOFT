# Rock Paper Scissors Game

## Description
This is a simple Rock Paper Scissors game developed using Python as part of the CodSoft Python Programming Internship (Task 4).

## Features
- User chooses Rock, Paper, or Scissors.
- Computer generates a random choice.
- Determines the winner based on game rules.
- Displays both user and computer choices.
- Keeps track of scores.
- Allows the user to play multiple rounds.

## Technologies Used
- Python 3
- random module

## How to Run
1. Clone this repository:
   git clone https://github.com/avinesh0904-creator/CODSOFT.git

2. Open the project folder.

3. Run the program:
   python rock_paper_scissors.py

## Game Rules
- Rock beats Scissors.
- Scissors beat Paper.
- Paper beats Rock.
- If both choose the same option, the result is a tie.
