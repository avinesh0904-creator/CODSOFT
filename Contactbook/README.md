# Contact Book

## Description
A simple Contact Book application developed using Python as part of the CodSoft Python Programming Internship (Task 5).

## Features
- Add new contacts
- View all saved contacts
- Search contacts by name or phone number
- Update existing contacts
- Delete contacts
- Easy-to-use menu-driven interface

## Contact Information Stored
- Name
- Phone Number
- Email
- Address

## Technologies Used
- Python 3

## How to Run

1. Clone the repository

git clone https://github.com/avinesh0904-creator/CODSOFT.git

2. Open the Contactbook folder.

3. Run the program

python contact_book.py
