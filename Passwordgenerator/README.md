# 🔐 Password Generator

## 📌 Project Overview
This is a simple Command-Line Password Generator developed using Python as part of the CODSOFT Python Programming Internship.

The application generates a strong and random password based on the length entered by the user.

## ✨ Features
- 🔢 User can choose the password length
- 🔐 Generates a random and secure password
- 🔡 Includes uppercase and lowercase letters
- 🔢 Includes numbers
- ✨ Includes special characters

## 🛠️ Technologies Used
- Python 3
- Visual Studio Code

## 📂 Project Structure
```
PasswordGenerator/
├── password_generator.py
└── README.md
```

## ▶️ How to Run

1. Open the project in Visual Studio Code.
2. Open the terminal.
3. Navigate to the project folder:
   ```bash
   cd PasswordGenerator
   ```
4. Run the program:
   ```bash
   python password_generator.py
   ```
   or
   ```bash
   py password_generator.py
   ```
